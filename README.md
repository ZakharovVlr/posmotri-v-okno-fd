# Проект «Посмотри в окно»

Учебный проект, реализованный в рамках курса «Фронтенд-разработчик» от Яндекс Практикума. 

## О проекте
«Посмотри в окно» — это интерактивное веб-приложение, которое позволяет пользователям просматривать виды из окон разных городов мира. Проект направлен на отработку навыков работы с макетом в Figma, сложной сеткой (Flexbox и Grid), кастомными элементами форм и адаптивностью.

## Стек технологий
* **HTML5**: семантическая разметка, использование тегов `<template>`.
* **CSS3**: 
  - Гибкие раскладки (Flexbox, Grid).
  - Кастомная стилизация форм без использования сторонних библиотек.
  - Работа с псевдоклассами (`:hover`, `:active`, `:focus-visible`) и псевдоэлементами (`::after`, `::placeholder`).
  - Многострочное ограничение текста (`-webkit-line-clamp`).
* **JavaScript**: (предоставлен в рамках заготовки) логика фильтрации, поиска и подгрузки видео.

## Функциональность
- Загрузка и фильтрация карточек городов по времени суток.
- Смена активного видео при клике на карточку.
- Динамическая подгрузка контента по кнопке «Показать ещё».
- Визуальная индикация процесса загрузки (прелоадеры) и ошибок.
- Полная доступность навигации с клавиатуры (фокусные состояния).

## Как запустить проект
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ZakharovVlr/posmotri-v-okno-fd.git
   ```
2. Откройте файл `index.html` в браузере или используйте расширение Live Server в VS Code.
